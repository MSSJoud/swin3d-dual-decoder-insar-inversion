# Sample Data

`punjab_real_sample.h5` is a tiny real-data subset derived from the Punjab deformation stack used in the experiments.

It contains:

- `z`: deformation stack with shape `24 x 96 x 96`
- `time`: acquisition dates
- `lat`: 1D latitude coordinate
- `lon`: 1D longitude coordinate

This file is included only as a smoke-test example for the package workflow. It is intentionally small and should not be treated as a scientific release product.
